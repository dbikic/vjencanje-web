# Images Folder

Place your images here:

- **hero-image.jpg** - Main hero image (displayed at the top of the page)
- **about-image.jpg** - Image for the "Our Story" section

## Supported formats:
- .jpg / .jpeg
- .png
- .webp

## Recommended sizes:
- Hero image: 1200px width or larger
- About image: 800px width or larger

You can use any filename, just make sure to update the `src` attribute in `index.html` to match your filename.
